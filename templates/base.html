Welcome to the Lung Cancer Prediction Project

When the project is opened, the home page is displayed to the user.

The home page contains 5 options, including a signup page for new users to create an account, a login page for existing users to access their account, an "About Us" page to provide information about the project, a "How to Use" page to guide users on how to use the project, and a "Result" page that displays the accuracy of the machine learning model used for lung cancer prediction.

To use the project, a new user must sign up by filling in their required credentials, including a username and password, for the first time.

An existing user can log in directly by entering their username and password.

After logging in, the user is directed to the lung cancer prediction page, where they are prompted to enter specific parameters, such as age and other factors on a scale of 1 to 8, to predict the likelihood of lung cancer.

Based on the parameters entered, the machine learning model predicts whether the patient has lung cancer or not.

Users can view the history of previous predictions by accessing the "Result" page, which displays the parameters.
